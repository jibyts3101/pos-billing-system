# POS Billing System

A modern Point of Sale (POS) Billing System built with **Spring Boot**, **Spring Security (JWT)**, **MySQL**, and REST APIs.

The goal of this project is to build a production-ready POS application suitable for supermarkets, retail stores, pharmacies, and other businesses.

---

# Features

- JWT Authentication
- Role-Based Access Control
- Product Management
- Customer Management
- Billing & Invoice Generation
- Inventory Management
- Inventory Logs
- Bill Search APIs
- RESTful API Architecture
